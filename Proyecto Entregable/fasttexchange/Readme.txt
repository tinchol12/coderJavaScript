Crear un sitio web simulando un exchange de criptos. El usuario debe ser capaz de:

1- Registrarse
2. Logearse
3. Cargar saldo
4. Consultar cotizaciones en tiempo real (APIS)
5. Realizar operaciones desde el saldo en ARS con otras monedas
6. Invertir en diferentes FCI

7. Consultar Cartera de inversiones
8. Retirar saldo
9. Estimar ganancias en FCI durante un período de tiempo
10. Automatización de pruebas con Cypress.io
    - Login
    - Registrarse
    - Consular estado de las APIS
    - Generar una operacion Pesos = $Coin
    - Invertir en un FCI
    - Calcular rendimientos durante un período de tiempo
    - Retirar Saldo
    - Reporte de Casos ejecutados
